# [제이펍] 바이브코딩 번역서 자료

## 8장
### 8.1.1

- 프롬프트1(국문)

```
빨간 구체가 튀어오르는 애니메이션을 앱을 JavaScript로 구현하고 아티팩트 창에 보여줘. 중력 효과를 적용하고, 공이 움직일 때는 궤적이 보이게 하고, 공이 바닥에 닿을 때는 에너지가 줄도록 만들어줘.
```
- 프롬프트1(영문)

```
Please write a JavaScript app in an artifacts window that animates a bouncing red sphere. Leave a trail behind it. Add gravity and energy when it hits the floor.
```

- 프롬프트2(국문)

```
●	공을 초록색으로 만들고 크기를 3배 키워 줘.
●	공을 하나 더 추가해줘.
●	이걸 게임으로 만들 수 있을까?
●	이 앱에서 중력이 어떻게 작동하는지 간단히 설명해 줘.
```

- 프롬프트2(영문)

```
• “Make the ball green and make it 3x larger.”
• “Add another ball.”
• “Can we turn it into a game?”
• “Explain in simple terms how the app’s gravity works.”
```


- 프롬프트3(국문)

```
색깔 있는 조명이 비추는 정육면체를 화면에 보여주는 JavaScript 프로그램을 작성해줘. 슬라이더 바를 추가해 정육면체의 방향을 바꿀 수 있게 해줘. 
```


- 프롬프트3(영문)

```
Write a JavaScript program that shows a cube with a colored light source; create slider bars that can change orientation of the polygon.
```

- 프롬프트4(국문)

```
5년 간격으로, 지난 50년간 촬영된 사진 수(추정치)를 막대그래프로 시각화 해줘.
```

- 프롬프트4(영문)

```
Please generate a bar chart visualization of the number of photographs taken (estimated) in 5-year intervals for the last 50 years.
```

### 8.1.2

- 프롬프트(국문)

```
브라우저에서 플레이할 수 있는 간단한 플래피 버드Flappy Bird 스타일의 게임을 만들어 줘. 기본적인 스타일링으로 보기 좋게 만들어 줘.
```

- 프롬프트(영문)

```
Please create a simple Flappy Bird-like game that I can play in the browser. Make it look nice with some basic styling.
```

- 프롬프트(국문)

```
●	속도를 늦춰서 게임을 더 플레이하기 쉽게 만들어 줘.
●	구름을 추가해줘.
●	게임을 더 좋게 만들어 줘.

```

- 프롬프트(영문)

```
• “Make the game more playable by slowing it down.”
• “Add clouds.”
• For fun: “Make it better.”
```
